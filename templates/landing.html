Features

현장 영업에 필요한 모든 것

엑셀과 카톡으로 관리하던 영업을, 지도 위에서 체계적으로.

🗺️

지도 기반 영업관리

가맹점·거래처를 지도 위에 표시하고, 클러스터링·히트맵으로 영업 현황을 한눈에 파악하세요.

📋

방문 기록 추적

방문 날짜, 결과, 메모를 기록하고 캘린더에서 한눈에 확인. 미방문 매장 자동 알림.

📊

실시간 대시보드

전환율 퍼널, 주간 비교, 팀원별 통계를 자동 생성. 엑셀 리포트 없이 성과를 확인하세요.

👥

팀 협업

팀별 데이터 격리, 초대 링크, 역할 관리. 여러 팀이 같은 플랫폼에서 독립적으로 운영.

📱

모바일 최적화

PWA로 앱처럼 설치 가능. 현장에서 스마트폰으로 바로 기록하고 조회하세요.

📤

데이터 내보내기

엑셀, CSV, JSON으로 언제든 데이터를 백업하고 보고서를 만드세요. 엑셀 일괄 업로드도 지원.
